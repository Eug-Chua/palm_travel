# Palm Travel Website Redesign

## Proof of Concept Presentation

**Prepared for:** Palm Travel Pte Ltd
**Date:** January 2026

---

## Executive Summary

This proof of concept demonstrates a complete redesign of Palm Travel's existing website, transforming it from a basic HTML site into a modern, professional, and mobile-responsive web presence that better reflects the company's 35+ years of trusted service to Singapore's maritime community.

## Project Objectives

1. **Modernize the visual design** - Fresh, professional look aligned with current web standards
2. **Improve mobile experience** - Fully responsive across all devices
3. **Highlight core services** - Clear presentation of Marine, Corporate, and Leisure offerings
4. **Build trust** - Prominently display IATA, NATAS, and STB accreditations
5. **Maintain simplicity** - Static site with no backend dependencies for easy maintenance

---

## What's Included in This POC

### Pages Developed

| Page | Description |
|------|-------------|
| **Home** | Hero banner, services overview, trust badges, client testimonials, CTA |
| **Services** | Detailed breakdown of Marine Crew, Corporate Travel, and Visa services |
| **Leisure** | Corporate retreat packages, destinations (Bali, Phuket, Bintan, Vietnam) |
| **About** | Company history, timeline, credentials, and values |
| **Contact** | Contact form, office location, and business hours |

### Key Features Demonstrated

- Fully responsive design (mobile, tablet, desktop)
- Modern teal/green color palette reflecting Palm Travel's branding
- Sticky navigation with mobile hamburger menu
- Animated elements and smooth hover effects
- Professional imagery with brand-consistent overlays
- Official accreditation logos (IATA, NATAS, STB)
- Contact form with validation

---

## Technology Approach

| Technology | Purpose |
|------------|---------|
| HTML5 | Semantic, accessible markup |
| Tailwind CSS | Modern utility-first styling |
| Vanilla JavaScript | Lightweight interactivity |
| Google Fonts (Inter) | Clean, professional typography |

**Why this stack?**
- No complex build process required
- Easy to maintain and update
- Fast loading times
- No ongoing hosting costs for databases or servers
